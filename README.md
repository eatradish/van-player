## Usage

Build van-player:

```
$ cargo build --release
```

Use:

```
$ ./target/release/van [URL1] [URL2]
```

Press [=] to add volume
Press [-] ro reduce volume
Press [p] to pause/unpause song
Press [l] to control playlist
